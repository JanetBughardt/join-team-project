# 🤝 Join – Team Project

Während meiner Weiterbildung an der **Developer Akademie** habe ich gemeinsam mit einem Team die Aufgabenverwaltungs-Webanwendung **Join** entwickelt.

🔗 **Original Repository:** https://github.com/hld2705/Join


---

## 📖 Projektbeschreibung

Join ist eine moderne Kanban-Webanwendung zur Verwaltung von Aufgaben. Nutzer können Aufgaben erstellen, bearbeiten und organisieren sowie Kontakte verwalten und den aktuellen Projektstatus über eine übersichtliche Benutzeroberfläche verfolgen.

Das Projekt wurde im Team entwickelt und simuliert einen professionellen Entwicklungsprozess mit Git, GitHub und Feature-Branches.

---

## 👩‍💻 Mein Beitrag

Während des Projekts war ich verantwortlich für:

* 🔥 Einrichtung und Konfiguration von Firebase
* 📊 Entwicklung der Summary-Seite
* ❓ Entwicklung der Help-Seite
* 🔗 Zusammenführung einzelner Projektbestandteile
* 📱 Umsetzung des responsiven Mobile-Designs (CSS)
* 🌿 Zusammenarbeit im Team mit Git, GitHub und Feature-Branches
